# House Price Prediction using Machine Learning
-> Project Overview:       
                          
This project predicts house prices based on property characteristics such as square footage, number of bedrooms, bathrooms, neighborhood, and year built. Multiple machine learning regression models were trained and evaluated to identify the most effective approach for house price estimation.

-> Dataset:
Total Records: 50,000

> Features:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms
- Neighborhood
- Year Built

> Target Variable:
- House Price

-> Data Preprocessing:
- Checked for missing values and data consistency.
- Encoded categorical features such as Neighborhood.
- Performed feature selection and transformation.
- Split data into 80% training and 20% testing sets.

-> Machine Learning Models Used:
- Linear Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Regressor
- Random Forest Regressor
- Support Vector Regressor (SVR)
  
-> Technologies Used:
Python, Pandas, NumPy, Scikit-Learn, Matplotlib, Gradio

-> Project Workflow:  
- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
  
-> Deployment using Gradio:
> Key Highlights
- Trained and compared 5 machine learning regression models.
- Used a dataset containing 50,000 housing records.
- Developed an interactive Gradio web application for real-time predictions.
- Applied preprocessing and feature engineering techniques to improve model performance.
